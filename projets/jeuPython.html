Jeu en Python

Jeu Python

Tout d'abord, ce projet devait être réalisé en équipe de 3 personnes. Or, mes deux camarades ont arrêté la formation au milieu du premier semestre, j'ai donc dû le réaliser tout seul.

Lors de cette SAE, nous avons dû réaliser 4 jeux en python. Ces 4 jeux sont: Le Juste Prix, le Jeu des Allumettes, le Jeu des Multiples et enfin la chasse au trésor. J'ai décidé de rajouter une interface graphique pour le menu du jeu. Chaque jeu possède son interface dans le terminal. De plus, tous les jeux sauf le Jeu des Multiples possèdent un mode contre l'ordinateur ou un mode "laisser l'ordinateur jouer". Ce projet fut très enrichissant dans mon apprentissage de l'algorithme.

Compétences Techniques Acquises: Python

Savoir-être acquis: Collaboration, Répartion des tâches
