Déploiement

Lors de ce projet, nous avons dû, en groupe de 2, réaliser le déploiement de l'application de messagerie Element.

Pour ce faire, nous avons utiliser des machines virtuelles que nous avons utilisé comme serveur. Nous avions donc une machine matrix-synapse, une machine pour la base donnée, une machine pour element et enfin une machine servant de reverse proxy. Lors de ce projet, nous avons tout d'abord apris à créer et configurer une machine virtuelle, puis nous avons installer postgresql. Suite à cela, nous avons installer et configurer synapse pour qu'il puisse intéragir avec la base de donnée et enfin, nous avons télécharger et configurer element et nginx afin de transformer ce dernier en reverse proxy. Nous avons rencontré plusieurs problèmes qui ont finalement été résolu, notamment le fait de mettre les bonnes indentations et les bonnes informations dans le .yml de synapse. Nous avons dû réaliser des procédures pour documenter tout ce que l'on a fait et que l'on puisse le reproduire.

Compétences Techniques Acquises: Rédaction de procédure en markdown, Configuration de synapse & nginx & postgresql, utilisation de SSH et de Tunnel SSH.

Savoir-être acquis: Collaboration, Répartion des tâches
