Installation Fedora

Fedora

Lors de cette SAE, nous avons dû, dans un premier temps, composer des setup pour des clients données. Nous étions par groupe de 2 et nous avons eu: des ordinateurs de bureau pour une utilisation de traitement de texte et un serveur nas. Nous avons donc travaillé ensemble sur les deux setups et nous avons donc pu chacun exprimer nos points de vue et débattre. Suite à cette première partie, nous avons dû installer une session fedora, puis récupérer le fichier kickstarter, le modifier pour l'adapter à un utilisateur du langage C. Nous avons dû ajouter également des utilisateurs avec un mot de passe crypté dans le fichier Kickstarter. Ensuite, nous avons inséré ce fichier dans la clé bootable pour pouvoir tester notre fichier d'installation automatique. Enfin, nous avons créé 3 scripts bash: un permettant de réaliser une sauvegarde, un deuxième permettant de restaurer à partir de cette sauvegarde et un dernier permettant de créer des utilisateurs.
