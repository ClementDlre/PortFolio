Déploiement d'un ERP

Déploiement

Lors de ce projet, nous avons dû, en groupe de 2, réaliser le déploiement de l'ERP Odoo de manière automatisée à l'aide d'Ansible, Terraform et Microsoft Azure.

Ce projet a été réalisé à l'IUT afin d'appliquer ce que l'on a vu en cours. Il se base sur le projet de déploiement d'Odoo de manière automatisé de la seconde année de BUT Informatique. Nous avons dû réaliser l'ensemble de l'infrastructure à l'aide de machine virtuelle Azure que nous déployons grâce à Terraform. Le projet se compose de 3 machines: un reverse proxy, une base de donnée et un machine servant à héberger les instances odoo. Suite à cela, nous avons écrit des playbook Ansible afin d'installer et de configurer les machines. Puis nous avons automatisé l'ensemble des actions grâce à des scripts bash. Un script pour la configuration des machines et un script pour la création d'une instance Odoo.

Compétences Techniques Acquises: Ansible, Terraform, Microsoft Azure, Docker, Docker Compose, Nginx, Postgres, Script Bash

Savoir-être acquis: Perseverance, Collaboration, Répartion des tâches, Recherhce
