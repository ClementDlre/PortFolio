Tri en Python

Tri Python

Tout d'abord, ce projet devait être réalisé en équipe de 3 personnes. Or, mes deux camarades ont arrêté la formation au milieu du premier semestre, j'ai donc dû le réaliser tout seul.

Ce projet a été réalisé lors des SAE du premier semestre de BUT Informatique. Nous avons dû coder 5 programmes en python permettant de trier une liste d'entiers. Nous avons dû coder le tri par insertion, le tri par sélection, le tri à bulles, le tri fusion et enfin le tri rapide. De plus, lors de cette SAE, nous avons dû réaliser une animation/visualisation de chaque tri en temps réel. Pour ce faire, nous avons utilisé la librairie "matplotlib" afin de réaliser cette visualisation grâce à un graphique.

Compétences Techniques Acquises: Python

Savoir-être acquis: Collaboration, Répartion des tâches
